# ScholarSync

## Description
ScholarSync is a simple student planner that helps users stay organized by tracking assignments, due dates, and daily tasks. It includes to-do lists, quick notes, and a study timer to boost focus. The goal is to reduce stress and improve time management through an easy-to-use platform.

## Features
-Assignment tracker
-To-do list
-Notes section
-Study Timer

## Technologies
-Python / HTML / CSS
